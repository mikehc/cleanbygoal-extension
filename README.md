# CleanByGoal

Closes browser tabs once the task they represent is actually finished — a merged GitHub PR, a
closed GitLab MR, a Jira issue marked Done, an archived Trello card — and separately cleans up
duplicate tabs. Every check happens locally in your browser; nothing is ever sent to an external
server.

This repo hosts the extension's website and issue tracker:

- **Site:** https://mikehc.github.io/cleanbygoal-extension/
- **Privacy policy:** https://mikehc.github.io/cleanbygoal-extension/privacy.html
- **Report a bug or request a feature:** use the Issues tab above

The extension's source code is not in this repository.
